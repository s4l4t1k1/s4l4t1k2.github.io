HTML Basics

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It is used to structure content on the web and create links between pages.

  • Tags: HTML uses tags to define the structure of a web page. Tags are enclosed in angle brackets and come in pairs, with an opening tag and a closing tag.
  • Attributes: HTML tags can have attributes, which provide additional information about the element. Attributes are included in the opening tag and have a name and a value.
  • Elements: An HTML element is composed of an opening tag, the content of the element, and a closing tag.

CSS Basics

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. It is used to control the visual appearance of web pages and make them more engaging and attractive.

  • Selectors: CSS uses selectors to target specific HTML elements and apply styles to them. Selectors can be based on element type, class, or ID.
  • Properties: CSS properties define the style of an element. Properties include color, font size, margin, and padding.
  • Values: CSS values specify the settings for a particular property. Values can be keywords, numbers, or units.
